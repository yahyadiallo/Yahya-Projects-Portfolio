# Washington EV Narrative Visualization

## Project Overview

This project is a narrative focused visualization. The visual uses a heatmap to show which electric vehicle brands are most common in Washington’s top cities for EV registrations.

The main narrative of the visualization is that electric vehicle ownership is heavily focused in larger cities, especially Seattle, and that Tesla dominates registrations across most of these locations.

## Visualization 

<img width="1165" height="689" alt="EV-Data-Chart" src="https://github.com/user-attachments/assets/a658702e-51cb-43b8-94a9-f438adfdd61f" />

This project uses a **heatmap visualization**.

- Rows represent Washington cities
- Columns represent vehicle manufacturers
- Darker squares represent higher registration counts

The visualization was designed to communicate patterns quickly to a public audience while also highlighting the dominance of Tesla compared to other EV brands.
