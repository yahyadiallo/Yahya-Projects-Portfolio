## Featured Projects

### 1. Seattle Housing Affordability Analysis
**Tools:** D3.js, JavaScript, HTML/CSS

Investigated whether household income growth in Seattle has kept pace with rising housing costs between 2018 and 2023. The project combines public housing and income datasets to analyze affordability trends before, during, and after the COVID-19 pandemic.

**Key Skills:**
- Data Cleaning
- Data Transformation
- Data Aggregation
- Data Visualization
- Data Storytelling

**Repository:**

### 2. Washington EV Registration Analysis
**Tools:** D3.js, JavaScript, HTML/CSS

Analyzed electric vehicle adoption patterns across Washington State using public registration data. The project uses a heatmap visualization to compare EV registrations across major cities and manufacturers, highlighting geographic adoption trends.

**Key Skills:**
- Data Cleaning
- Exploratory Data Analysis
- Heatmap Visualization
- Data Storytelling
- Public Data Analysis

**Repository:**
[Washington EV Registration Analysis](./washington-ev-registration-analysis)
